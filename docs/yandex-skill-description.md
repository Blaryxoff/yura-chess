# Yandex skill copy

## Name

Шахматы с Юрой

## Short description

Полноценные шахматы с сильным Stockfish, которыми можно управлять только голосом. Навык понимает ходы, объясняет ошибки и помнит партии.

## Full description

«Шахматы с Юрой» — голосовой шахматный партнёр для полноценной игры против Stockfish. Навык создавался прежде всего для незрячего опытного шахматиста, поэтому всю партию можно провести без экрана и кнопок.

Называйте ходы обычными словами: «пешка е два е четыре», «конь эф три», «короткая рокировка». Если фраза неоднозначна, Алиса уточнит ход и не изменит позицию. Если ход невозможен, навык объяснит причину: фигура не может так ходить, путь перекрыт, поле занято или король останется под шахом.

Во время партии можно спросить:

- «что на эф три»;
- «где белые слоны»;
- «какая позиция»;
- «чей ход»;
- «что сделали чёрные четыре хода назад»;
- «повтори медленно».

Незаконченную партию можно продолжить позже: навык найдёт её и напомнит последние два хода. Можно выбрать цвет и уровень силы движка. На телефонах и других устройствах с экраном дополнительно показывается доска с координатами и последним ходом, но вся необходимая информация всегда доступна голосом.

## Example requests

- Начать игру чёрными, уровень десять
- Пешка е два е четыре
- Где белые слоны
- Что на эф три
- Какая позиция
- Что сделали чёрные четыре хода назад
- Продолжить последнюю партию

## Privacy note

Для сохранения партий используется обезличенный идентификатор. Чтобы улучшать распознавание команд, навык временно хранит нормализованный текст фразы и результат её обработки. Аудиозаписи, токены и прямой идентификатор пользователя не сохраняются.
