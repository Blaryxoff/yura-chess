# Шахматы с Юрой

«Шахматы с Юрой» — голосовой шахматный навык для Яндекс Алисы. Он создавался в
первую очередь для незрячих и слабовидящих игроков, но подходит всем, кому
удобнее играть и разбирать партии голосом.

Навык понимает естественные русские шахматные фразы, объясняет недопустимые ходы,
описывает позицию, сохраняет партии и играет с помощью Stockfish. Все обязательные
действия доступны без экрана; на устройствах с дисплеем доска показывается только
как дополнение к полному голосовому ответу.

Сайт проекта: [chess.waxim.ru](https://chess.waxim.ru)

## Как пользоваться

Команды не нужно запоминать дословно: навык распознаёт разные естественные
формулировки. В любой момент можно сказать «что ты умеешь», «помощь» или
«как играть». Команды «дальше», «назад» и «сначала» помогают перемещаться по
голосовой справке.

| Задача                       | Примеры команд                                                                        |
|------------------------------|---------------------------------------------------------------------------------------|
| Начать или продолжить партию | «новая игра белыми уровень пять», «новая игра черными», «продолжить последнюю партию» |
| Сделать ход                  | «пешка е два е четыре», «конь эф три», «е два е четыре», «рокировка»                  |
| Узнать положение на доске    | «какая позиция», «что на е четыре», «где белые слоны», «чей ход», «есть ли шах»       |
| Узнать о партии              | «какой был последний ход», «какие фигуры съедены», «какой дебют», «за кого я играю»   |
| Настроить ответы             | «говори кратко», «говори подробнее», «говори медленнее», «короткая нотация»           |
| Включить помощь тренера      | «включи режим тренера», «оцени позицию», «какие ходы хорошие», «подскажи»             |
| Разобрать сыгранную партию   | «разбери партию», «где перелом», «главная ошибка», «продиктуй ходы»                   |
| Решать задачи                | «дай задачу», «задача на мат в один», «покажи решение», «какая у меня серия»          |
| Исправить недопонимание      | «что ты услышала», «повтори медленно», «повтори координаты по буквам»                 |

Если фраза допускает несколько ходов или распознана неуверенно, навык просит
уточнение и не меняет позицию. Если ход недопустим, навык объясняет конкретную причину.

Настройки пауз, шахматной нотации, ориентации доски и режима
следующей партии сохраняются для игрока между сессиями. Подсказки и оценки
Stockfish доступны только в режиме тренера: обычная партия остаётся честной.

## Что есть в проекте

- Полная партия с сохранением состояния и продолжением в новой сессии Алисы.
- Голосовое описание доски, фигур, шахов, рокировки, истории ходов и фактов о партии.
- Уровни сложности Stockfish и ограниченный пул постоянно запущенных процессов движка.
- Режим тренера с оценкой позиции, кандидатами, угрозами и объяснением ходов.
- Разбор ошибок в партии, переломных моментов, истории и экспорт PGN.
- Голосовые шахматные задачи и сохранение серии решений.
- Изображение доски для устройств с экраном.
- Псевдонимная статистика без хранения исходного идентификатора пользователя Алисы.
- Консольный интерфейс для разработки и проверки сценариев без устройства с Алисой.

## Как устроен проект

Это модульный монолит на Python 3.12, FastAPI, SQLAlchemy и MariaDB. Шахматные
правила проверяет `python-chess`; Stockfish только выбирает ход и анализирует
позицию, но никогда не решает, допустим ли ход игрока.

| Каталог                        | Ответственность                                      |
|--------------------------------|------------------------------------------------------|
| `src/yura_chess/adapters/`     | Протокол Алисы и внешние API Яндекса                 |
| `src/yura_chess/application/`  | Сценарии приложения и маршрутизация команд           |
| `src/yura_chess/domain/`       | Состояние партии, настройки, разбор и задачи         |
| `src/yura_chess/voice/`        | Нормализация речи, распознавание и диагностика ходов |
| `src/yura_chess/storage/`      | Модели и репозитории MariaDB                         |
| `src/yura_chess/engine/`       | Ограниченный пул процессов Stockfish                 |
| `src/yura_chess/presentation/` | Голосовые ответы, PGN, доска и веб-страница          |

Полная история ходов в формате UCI — основной источник состояния партии; FEN
вычисляется из неё. Повторная доставка одного запроса Алисы идемпотентна и не
может привести ко второму ходу.
